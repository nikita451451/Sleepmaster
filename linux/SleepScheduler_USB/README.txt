Sleep Scheduler - установка без интернета

Инструкция:
1. Вставьте флешку в Linux-компьютер
2. Откройте терминал
3. Перейдите на флешку:
     cd /media/USERNAME/FLASH_NAME/SleepScheduler_USB
4. Дайте права:
     sudo chmod +x install.sh
5. Запустите:
     sudo ./install.sh

Готово! Для запуска используйте `sleep-scheduler`
